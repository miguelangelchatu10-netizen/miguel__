# Proyecto-Web-Ejemplo

Este proyecto consiste en el diseño de un sitio web implementando HTML5 y CSS3, enfocado en buenas prácticas de maquetación y diseño responsivo. Sirve como demostración práctica para la gestión de ramas, control de versiones y resolución de conflictos utilizando Git y GitHub.
